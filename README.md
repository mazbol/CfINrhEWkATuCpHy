# Nanosia Justplay

A progressive web application featuring movies and tv shows from the [The Movie Database (TMDb)](https://developers.themoviedb.org/3) API. This web application is dedicated to landing pages as online marketing for CPA businesses. Built using [Vue.js](https://github.com/vuejs/vue) and [Nuxt.js](https://github.com/nuxt/nuxt.js).

## Quick setup

1. Install [Node](https://nodejs.org/) - *We recommend you have the latest LTS version installed.*
2. Edit file `mopie.js` change replace `https://nanosia.com/` with your offer url
3. To put an *analytic* script or *histats* please edit file `app.html` and put it on `<!-- put here your code -->`
4. Open a *terminal* or *cmd*, and run the following command.

## Build Setup

```bash
# install dependencies
$ npm install

# generate static project
$ npm run generate
```

After running the command above, there will be a folder `dist`.
The `dist` folder is ready to be uploaded on your hosting, or on a hosting service provider for static sites, for example `netlify`

### To run on localhost
```bash
# serve at localhost:3000
$ npm run start
```

### Development
```bash
# serve with hot reload at localhost:3000
$ npm run dev
```
